# Tivic Bank App

## Sistema que replica as ações de um sistema bancário simples.

### Página Inicial
![image](https://user-images.githubusercontent.com/31075618/231566075-e0ba1412-eb29-4d80-a2a0-6665e4660fcd.png)

### Para iniciar qualquer movimentação é necessário que possua cadastro no sistema. Para cadastrar clique em Correntistas localizado a direita.
![image](https://user-images.githubusercontent.com/31075618/231565899-95282829-17ee-4276-b28c-5e8d54f24c42.png)

### Na página correntista, você vai cadastrar o CPF e o nome do novo correntista e clica em gravar. O sistema vai gerar um número de conta e cadastrar o correntista com saldo R$0, como mostra a imagem abaixo. Clicando em voltar você é redirecionado a página de Movimentações.
![image](https://user-images.githubusercontent.com/31075618/231568220-6d67d32f-c0b6-41d1-a176-b9043206d403.png)

### Depois de cadastrado já é possível encontrar o novo correntista na lista de correntistas cadastrados.
![image](https://user-images.githubusercontent.com/31075618/231571221-4e2dc573-bd7d-4a95-9933-30370fea887d.png)

### Para iniciar uma movimentação, clique em Nova Movimentação.
![image](https://user-images.githubusercontent.com/31075618/231571817-f1114b05-bdfb-4b6f-9eb3-47fb7aaf1736.png)

### Nesta página você irá selecionar a data da movimentação, o correntista da conta movimentada, selecionar o tipo de movimentação e o valor da movimentação. O campo Descrição é opcional. Ao clicar em confirmar, os dados serão enviados e a página será atualizada. Clicando em voltar o sistema redicionará para a página inicial.
![image](https://user-images.githubusercontent.com/31075618/231575192-d2406fc7-a138-49d2-962b-5e5b4509b014.png)

### Ao selecionar um correntista, o sistema apresentará as movimentações que ele realizou.
![image](https://user-images.githubusercontent.com/31075618/231582121-d253d909-7287-4b75-92e7-c99f41868492.png)
